AI Engine

Dual-model
intelligence.

Two state-of-the-art models work in sequence — each doing what it does best — for results no single model can match.

Object Detection

YOLOv8

Ultralytics / Real-time

You Only Look Once — the gold standard for real-time object detection. YOLOv8 locates the waste object in the frame with a bounding box, then hands it off for deep classification.

  • Single-pass inference under 20ms
  • Anchor-free detection head
  • Optimized for edge deployment
  • Precision bounding box output
Image Classification

LeViT-256

Vision Transformer / Fine-tuned

A hybrid Vision Transformer that blends attention mechanisms with efficient pooling. Fine-tuned on 9 waste categories, it achieves 90.14% accuracy with a compact architecture suited for real-time systems.

  • 90.14% top-1 classification accuracy
  • Attention + pooling hybrid layers
  • 9-class fine-tuned waste model
  • Efficient transformer architecture
